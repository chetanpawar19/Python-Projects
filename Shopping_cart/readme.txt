Project name- Shopping cart backend
Author- Chetan Pawar

Details:

The shopping cart backend project allow to browse various items, select them as per quantity, add them in cart, cart subtotal & grand total.
The project is created to demonstrate the use of python classes, inheritance, module, packages etc.

Requirements:
Python 3.0 and above

Steps:
-Opem Main.py file in python IDLE or any and run the file.
OR 
-Open command prompt, navigate to Main.py file path, type command-
"python Main.py" and hit enter
